# DownReels - Instagram Downloader

A modern, mobile-first web app to download Instagram Reels, Videos, Stories, Photos and Audio.

## Setup
1. Get a free RapidAPI key for the Instagram Downloader API.
2. Replace `YOUR_RAPIDAPI_KEY` in `assets/js/config.js`.
3. Open `index.html` in a browser or deploy the folder to any static host (GitHub Pages, Netlify, Vercel).

## Features
- Paste any Instagram link and download instantly
- No watermarks, HD quality
- Works on all devices
- Clean dark UI with glassmorphism
